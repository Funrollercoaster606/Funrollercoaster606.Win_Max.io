ABOUT WINDOWS MAX

To see info about our releases, check our releases and news page.

Windows Max is a solo project about modified windows developed for gaming. With Windows Max, we install extra features and make the operating system run better.

That doesn't mean Windows Max isn't for buisness or home use, it can be used for that too! just install 1.0.0 compact for free!

Windows Max is totally free, too! Windows Max intends to make no profits.
